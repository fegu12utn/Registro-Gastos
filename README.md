# 💰 Registro de Gastos

Aplicación desarrollada en React que permite gestionar un registro de gastos personales.


## 🚀 Funcionalidades

- Listar gastos desde un backend simulado
- Agregar nuevos gastos mediante un formulario
- Eliminar gastos de la lista
- Calcular el total gastado
- Mostrar el gasto más alto
- Filtrar gastos por categoría


## 🛠️ Tecnologías utilizadas

- React
- JavaScript
- Hooks (useState, useEffect)
- Axios
- JSON Server (backend simulado)

## ▶️ Cómo ejecutar el proyecto

1. Clonar el repositorio
2. Instalar dependencias:

```bash
npm install
npm run dev
```
